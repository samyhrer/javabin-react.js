How?

Core design principle:

Re-render the whole application on every update

  • Touching the DOM is slow
  • Loses form state
  • Loses scroll position

Solution: The virtual DOM

On every update...

  1. React builds a new virtual DOM subtree
  2. ...diffs it with the old one
  3. ...computes the minimal set of DOM mutatitions and batch executes all updates

Building blocks: React Components

A simple React Component


var HelloMessage = React.createClass({
  render: function() {
    return React.DOM.div(null, "Hello ", this.props.name);
  }
});
React.renderComponent(HelloMessage( {name:"JavaBin"} ), document.body);

using JSX


var HelloMessage = React.createClass({
  render: function() {
    return <div>Hello {this.props.name}</div>;
  }
});
React.renderComponent(<HelloMessage name="JavaBin" />, document.body);
